# Scope

## P0

- Startup and recovery stability.
- History searchability across model, LoRA, prompt tags, rating, Pro Recipe notes, favorite/candidate/rejected/asset labels.
- Download and Model Library cleanup for stale jobs, orphan partials, and model profile relationships.

## P1

- Prompt Composer slot templates and model-specific prompt guidance.
- Candidate Board review and variation actions.
- Reference Board for source/reference images.
- Upscale comparison and adopted setting capture.

## Non-goals

- Public distribution work, code signing, auto-update metadata, and general release polish.
- Automatic deletion or process termination where a false positive could damage active runtime state.
- Replacing Forge/Gradio internals or embedding the Forge UI.
