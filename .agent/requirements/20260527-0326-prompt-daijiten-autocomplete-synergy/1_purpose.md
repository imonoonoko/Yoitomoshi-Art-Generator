# Purpose

Prompt Daijiten should move from "large tag search" toward a practical prompt command surface:

- Broaden tag coverage with source-governed public vocabulary metadata.
- Make autocomplete easier to judge at the moment of insertion.
- Make existing Yoitomoshi features visible as a connected production system.
- Leave a durable HTML report that shows feature inventory, synergy opportunities, evidence, and next steps.

Primary user:

- A local Stable Diffusion / Forge power user who repeatedly reuses prompts, histories, models, LoRA, references, upscale comparisons, and tag review data inside this app.
