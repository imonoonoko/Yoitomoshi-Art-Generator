  • CharacterCompositeIntegrationStatus を共有型に追加。

UI

  • AIキャラ追加パネルに連携状態カードを追加。
  • 再確認ボタンで状態をリフレッシュ可能。
  • ja/en/ru/ptの文言を追加。
