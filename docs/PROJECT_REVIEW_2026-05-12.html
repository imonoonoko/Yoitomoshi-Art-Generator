機能別精査

生成

txt2img / img2img

モデル選択、VAE、sampler/scheduler、seed、LoRA、Prompt Helper、Prompt Library、履歴復元が揃っている。Generateボタンはタブ状態を明示し、txt2img/img2img以外では誤生成しない設計。

ControlNet

Builder + 通常Panel

Builderは役割別プリセット、画像ドロップ、preprocessor実行、source / preprocessedプレビュー、Unit 1反映まで対応。通常ControlNet Panelは最大3ユニットを維持。

Upscale

Simple / Diffusion / Ultimate

Tile ControlNet専用トグル、metadata-driven suggestion、Tile ON/OFF × denoise 0.25/0.35/0.45 の比較生成、判断基準付き保存に対応。残る確認は実画像での採用基準確定。

Tools

Workspace / Startup / Library / Merger / Converter

Toolsは開閉式で重い処理を遅延。Workspace保存は embed / references / settings-only、Model Libraryは復旧、Health、Merger、Converterの導線がある。Converterは対象ファイル待ち。
