全体精査レポートを元にしたブラッシュアップ

全体機能・シナジー精査レポートでP0に置いた 「Recipe保存の統一」「History整理とFABRICの接続」「生成前の失敗予防」を、既存UIに追加実装した。

typecheck PASS build PASS 2026-05-13 Electron / React / Forge

実装サマリー

Recipe保存の統一

ToolsのWorkspace保存とPreviewのRecipe保存が、共通の buildWorkspaceSnapshot() を使うようになった。 保存漏れが起きやすかった拡張設定も同じ範囲で扱う。

HistoryからFABRICへ

履歴ラベルの「お気に入り / 採用候補 / 素材化済み」をFABRIC positiveへ、 「没」をnegativeへ送れるようにした。重複投入は history:{id} で防ぐ。

生成前チェック

PromptPanelにPreflightを追加し、Forge状態、モデル選択、img2img画像、 ControlNet、FABRIC、ADetailerなどの詰まりやすい条件を生成前に表示する。

生成失敗予防 Generateボタンが押せない理由や、ControlNet画像なしなどの警告が各パネルに分散していた。 PromptPanel下部にPreflightを常時表示。停止条件と注意条件を生成前に確認できる。
