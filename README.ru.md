# Yoitomoshi Art Generator

[日本語](README.md) | [English](README.en.md) | **Русский** | [Português](README.pt.md)

Личный фронтенд для Stable Diffusion, созданный для разработки игр. В фоне запускает Stable Diffusion WebUI Forge и предоставляет генерацию, управление ассетами и интеграцию с Civitai через интерфейс на React/Electron.

> **Быстрый старт**: следуйте разделу «Запуск — 3 шага» ниже. Подробное руководство по настройке сейчас ведётся на японском: [`docs/SETUP_GUIDE.md`](docs/SETUP_GUIDE.md).

## Запуск с GitHub — 4 шага

### 1. Получить репозиторий

Если установлен Git:

```powershell
git clone https://github.com/imonoonoko/Yoitomoshi-Art-Generator.git
cd Yoitomoshi-Art-Generator
```

Без Git используйте `Code` → `Download ZIP` на GitHub и распакуйте папку.

`runtime/`, `userdata/`, `node_modules/` и результаты сборки не входят в репозиторий. Forge и модели нужно подготовить локально.

### 2. Что нужно

| Обязательно | Назначение |
|---|---|
| **Stable Diffusion WebUI Forge** | Уже работает (`run.bat` запускается) — [страница релизов](https://github.com/lllyasviel/stable-diffusion-webui-forge) |
| **Node.js 22.x (LTS)** | Среда выполнения приложения — установите LTS-сборку с [nodejs.org](https://nodejs.org/) |
| **Windows 10/11** | Проверенная платформа (Mac/Linux не тестировались) |
| **NVIDIA GPU** | Проверено на RTX 4060 Ti 8GB. Для SDXL рекомендуется `--medvram`. |

Требуется около 5 ГБ свободного места (node_modules + история сгенерированных изображений).

### 3. Запуск двойным щелчком

```
Дважды щёлкните  Yoitomoshi.bat  в проводнике
```

При первом запуске автоматически выполняется (3–5 минут):

1. `npm install` — загрузка зависимостей
2. `npm run build` — сборка приложения

После сборки откроется окно Electron. Последующие запуски — за секунды.

> **Иконка на рабочем столе**: щёлкните правой кнопкой по [`create-desktop-shortcut.ps1`](create-desktop-shortcut.ps1) → «Запустить с PowerShell», и на рабочем столе появится ярлык.

### 4. Первоначальная настройка — путь к Forge

Нажмите ⚙ в правом верхнем углу строки заголовка, чтобы открыть окно настроек:

| Поле | Описание |
|---|---|
| **Путь к установке Forge** | Абсолютный путь к родительской папке с `run.bat`. По умолчанию: `C:\宵灯工房アート\Yoitomoshi-Art-Generator\runtime\forge` |
| **Порт Forge** | По умолчанию `7860` (измените, если запускаете Forge на другом порту) |
| **Автозапуск** | Если включено, Forge стартует вместе с Electron |
| **API-ключ Civitai** | Необязательно. Нужен для NSFW-моделей и повышенного лимита запросов. Создайте на вкладке «API Keys» в [Civitai → Account](https://civitai.com/user/account). |

После сохранения нажмите кнопку питания слева в строке заголовка — Forge запустится в фоне (1–2 минуты при первой инициализации зависимостей).

## Где хранятся данные

Приложение портативное. Всё перечисленное лежит в `userdata/` рядом с проектом:

```
userdata/
├── settings.json              настройки (путь к Forge, ключ API, …)
├── presets.json               пресеты промптов
├── quick-presets.json         пользовательские быстрые пресеты
├── hidden-quick-presets.json  ID скрытых встроенных пресетов
├── favorites.json             избранные теги
├── lora-favorites.json        избранные LoRA
├── lora-usage.json            история использования LoRA (для авто-подсказок)
├── custom-prompt-library.json пользовательские категории/теги
├── civitai/                   кэш метаданных Civitai
│   ├── <sha256>.json          по чекпоинту
│   ├── lora-<sha256>.json     по LoRA
│   ├── community-<id>.json    агрегаты по сообществу
│   ├── update-check.json      проверка обновлений (TTL 24 ч)
│   └── tags.json              кэш популярных тегов (TTL 24 ч)
└── history/                   история генераций
    ├── index.json
    └── <uuid>.png             каждое изображение (макс. 500)
```

Перенесите **всю папку проекта** на другой диск или ПК — настройки, история и кэш переедут вместе.

## Решение проблем

### Браузер открывается сам
Перед запуском приложение перезаписывает `webui/config.json` Forge, выставляя `auto_launch_browser` в `Disable`. Если браузер всё равно открывается — Forge запускается каким-то иным путём. Проверьте, что в поле «Forge extra args» в настройках нет, например, `--api`.

### Не появляется «Running on local URL»
- Проверьте путь к Forge (должен существовать `<path>/webui/launch.py`)
- Конфликт портов: другой Forge или приложение уже занимает 7860
- При первом запуске обновление зависимостей Forge может занять несколько минут

### «Не PNG-изображение» при разборе метаданных
Картинки, прошедшие через соцсети/CDN, часто теряют EXIF. Используйте кнопку «Парсинг из текста» и вставьте параметры в формате A1111 или формате «label + newline + value».

### Те же ошибки расширений повторяются
Нажмите ⚠️ на строке заголовка → «Отключить» — расширение добавится в `disabled_extensions` в `webui/config.json` Forge. Перезапустите Forge для применения.

### Рекомендации модели устарели
Кэш — `userdata/civitai/<sha>.json`, TTL 14 дней. Удалите файл, чтобы запросить заново при следующем выборе. То же для агрегатов сообщества (`community-<id>.json`).

## Известные ограничения

- **Узкий круг распространения**: создавалось для личного пользования, не предназначено для широкой аудитории
- **NSFW-фильтр только в поиске Civitai**: сама генерация не модерируется (это на стороне Forge / модели)
- **Метаданные PNG**: поддерживается только формат A1111 / Forge (чанк `parameters` / EXIF UserComment)
- **Mac/Linux не тестировались**: код кроссплатформенный, но запуск Forge ориентирован на Windows
